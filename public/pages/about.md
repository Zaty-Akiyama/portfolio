大学では物理学を専攻し、卒業後の2022年4月にITの個人事業を開始し、フリーランスとして活動を始めました。
主に、**Web**サイトのフロント／バックエンドの設計・開発コーディング・運用保守を行っています。

現在は、React／Next.jsのWebサイト制作や、Vue／Nuxt.jsを用いたWebアプリケーション開発、各種システムで用いるRest APIの設計・開発を中心に活動しています。

## スキルセット

プロジェクトに応じて、必要な技術を柔軟に取り入れていますが、主に以下の技術スタックでの開発経験が豊富です。

### フロントエンド

- **言語**: JavaScript, TypeScript, HTML5, CSS3, SCSS
- **フレームワーク／ライブラリ**: React, Next.js, Vue, Nuxt.js
- **スタイルライブラリ**: Tailwind

### バックエンド

- **言語**: PHP
- **フレームワーク**: Laravel
- **データベース**: MySQL
